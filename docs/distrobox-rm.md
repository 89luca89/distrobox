# distrobox-rm

## DESCRIPTION

distrobox-rm deletes one of the available distroboxes.

Usage: distrobox-rm **\--name** container-name \[\--force\]

## OPTIONS

**\--name**/-n:

:   name for the distrobox

**\--force**/-f:

:   force deletion

**\--help**/-h:

:   show this message

**\--verbose**/-v:

:   show more verbosity

**\--version**/-V:

:   show version
