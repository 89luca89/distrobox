# distrobox-list

## DESCRIPTION

distrobox-list lists available distroboxes. It detects them and lists
them separately from the rest of normal podman or docker containers.

Usage: distrobox-list

## OPTIONS

**\--help**/-h:

:   show this message

**\--verbose**/-v:

:   show more verbosity

**\--version**/-V:

:   show version
